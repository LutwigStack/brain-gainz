# 01 Manual Check Flow

## Status

`planned`

## Goal

Make manual checks clear in the learner UI.

## Scope

- review manual check copy
- verify what the learner must enter
- verify what proof/result is required
- remove or hide developer-like labels from the main surface
- test failed and passed attempts

## Done When

- user can tell why the action is disabled
- passed and failed manual attempts are understandable
- browser QA has screenshots or notes
