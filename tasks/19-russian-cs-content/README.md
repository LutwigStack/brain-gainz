# 19 Russian CS Content

## Status

`done`

## Goal

Make the CS campaign feel Russian-first for the learner, not only for the app shell.

The previous Russian pass improved controls and common UI. This epic translates and normalizes the actual learning content: campaign names, route names, node titles, descriptions, tasks, checks, and daily text.

## Scope

Includes:
- CS campaign names and visible labels
- route/stage names
- node titles and descriptions
- daily tasks
- check prompts and expected answer hints
- Wind Rose branch text
- route overview text
- glossary for stable Russian terms

Excludes:
- translating internal code identifiers
- changing database keys only for style
- translating third-party library names
- rewriting the whole course academically

## Success Criteria

- A learner can use the CS campaign without reading English UI or course instructions.
- Core terms are consistent: проверка, маршрут, узел, прогресс, подтверждено, повторить.
- Check prompts and answers still work after translation.
- Screenshots of Today, Map, Route, Checks, and Wind Rose are Russian-first.
- Tests, lint, and build pass.
