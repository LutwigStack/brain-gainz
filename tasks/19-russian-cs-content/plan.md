# Russian CS Content Plan

## Order

1. Define a small Russian glossary for product and course terms.
2. Translate campaign, route, stage, stat, and branch text.
3. Translate node titles, descriptions, and daily task text.
4. Translate check prompts and verify answers still pass.
5. Run browser QA across the CS campaign.

## Main Risk

Changing visible text can break exact checks if expected answers are translated inconsistently.

- Today
