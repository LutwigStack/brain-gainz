# Plan

## Execution Order

1. Упростить shell до двух основных вкладок и ужать `Сейчас` до digest-экрана.
2. Пересобрать `Карта` как главный workspace: summary bar сверху, большая карта слева.
3. Добавить правый rail с раскрывающимся editor и постоянным summary выбранного узла.

## Guardrails

- Не оставлять старую конкуренцию между `Сейчас` и `Карта` как двумя равноправными рабочими экранами.
- Не превращать `Сейчас` обратно в action dashboard под новым названием.
- Не смешивать node editing и node summary в одну неразличимую панель.
- Не делать editor настолько большим, что он вытесняет map surface.
- Не тащить весь workflow редактирования связей в первый pass, если для него нужен отдельный interaction model.

## Definition Of Done For Epic

- app shell и product narrative опираются на две основные вкладки;
- `Сейчас` даёт короткую и полезную сводку по трем секциям;
- `Карта` является главным экраном работы с узлами;
- правый rail стабильно делится на `Node Editor` и `Node Summary`;
- task system отражает новый UX-поворот как отдельный rollout, а не как размытое продолжение старого epic.
