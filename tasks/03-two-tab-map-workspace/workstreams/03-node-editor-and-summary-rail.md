# 03 Node Editor And Summary Rail

## Status

`done`

## Goal

Добавить в правый rail `Карта` две разные, но связанные секции:
- раскрывающийся `Node Editor`
- постоянный `Node Summary`

## Why

`Карта` уже стала основным workspace, но правый rail раньше не был разделен на режим редактирования и режим чтения состояния. Без этого ручное ведение узлов оставалось неполным.

## Scope

- верхняя editor-секция в правом rail
- поля ручного редактирования узла
- сохранение, дублирование, архивирование
- нижняя summary-секция с прогрессом, задачами, unlocks и next-step
- разведение editor и summary по ролям и визуальной иерархии

## Done When

- в правом rail есть отдельный editor и отдельный summary
- editor не вытесняет карту
- summary остается читаемым companion-pane
- ручное заполнение узлов становится first-class workflow прямо из `Карта`
