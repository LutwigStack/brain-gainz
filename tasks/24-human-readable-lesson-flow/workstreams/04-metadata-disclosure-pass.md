# 04 Metadata Disclosure Pass

## Status

`planned`

## Goal

Move check metadata out of the primary learner surface.

## Scope

- hide or collapse check method details
- keep author/debug details in author mode or disclosure
- keep learner-facing explanation for why the answer passes or fails
- ensure manual and AI-assisted checks still explain what evidence is needed

## Done When

- learner sees learning criteria, not implementation details
- author mode still has enough information to edit checks
- no assessment behavior changes unexpectedly
