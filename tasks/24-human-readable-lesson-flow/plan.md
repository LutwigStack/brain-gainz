# Human Readable Lesson Flow Plan

## Order

1. Replace technical learner copy with study-language copy.
2. Compress the lesson header into task, criteria, and action.
3. Make task prompt and criteria visually stronger than metadata.
4. Move check metadata into secondary detail.
5. Review mobile lesson readability.
6. Run browser QA.

## Main Risk

Do not hide information that explains why a check passes or fails. Hide technical implementation details, not learning criteria.
