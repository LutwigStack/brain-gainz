# 04 Map Stats And Balance

## Status

`done`

## Goal

Keep the larger CS campaign readable and balanced.

## Scope

- map overview readability
- route overview grouping
- Wind Rose/stat coverage
- task selection balance
- weak spot volume
- route progress counters

## Done When

- larger graph does not become unreadable
- route overview still shows current step and branch progress
- Today does not drown the user in too many choices
- statistics remain useful

## Result

Route overview groups the expanded CS route into five stages and shows Database Systems as a compact stage card. Wind Rose includes the new Databases stat without pushing the existing stats out of the learner view.
