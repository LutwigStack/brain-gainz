# 08 Campaign Now View

## Status

`planned`

## Goal

Make `Now` act as a compact campaign-specific next-action surface.

## Scope

- current campaign focus
- next useful step
- weak stat signal
- active branch signal
- quick jump to map
- quick jump to wind rose stat

## UX Direction

- The user should understand the next action in a few seconds.
- Avoid long recommendations.
- Use compact visual state:
  - current branch
  - stat marker
  - progress
  - one primary action

## Done When

- `Now` only reads selected campaign data
- it shows one clear next action
- it can route to the relevant map or stat
- it does not become a text feed
