# 03 Mobile Navigation Clarity

## Status

`done`

## Goal

Make mobile navigation understandable before the user memorizes icons.

## Scope

- review 390px learner path navigation
- avoid an unexplained row of icon-only controls on first use
- show compact labels for the most important destinations
- keep secondary destinations in an overflow or lower-priority area
- keep header height under control
- preserve desktop navigation behavior

## Done When

- mobile first path is understandable from labels and states
- main lesson action appears early enough
- no document-level horizontal overflow
- screenshots cover Campaign, Today, and Lesson
