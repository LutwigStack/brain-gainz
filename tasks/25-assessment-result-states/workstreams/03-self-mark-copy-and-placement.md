# 03 Self Mark Copy And Placement

## Status

`planned`

## Goal

Rename and reposition self-mark so it reads like a learner choice, not a technical mode.

## Problem Copy

`Сам отметил без XP`

## Suggested Copy

- primary label: `Отметить для себя`
- helper: `Без зачета и XP`
- optional result copy: `Сохранено как личная отметка`

## Scope

- update learner-facing copy
- keep self-mark secondary to real verification
- make difference from confirmed progress clear
- avoid changing XP/mastery logic

## Done When

- learner can tell self-mark is not verified mastery
- wording is short enough for mobile
- tests cover copy if existing copy tests apply
