# 01 CS Bachelor Curriculum Scope

## Status

`planned`

## Goal

Define a realistic simplified `Computer Science Bachelor` campaign outline that can drive product testing without pretending to be a complete accredited curriculum.

## Scope

- draft the full high-level curriculum map by years/stages
- define branches, prerequisites, and expected mastery outcomes
- choose the first playable slice
- identify which nodes need checks and which can remain practice/navigation nodes
- define route stages for the first specialization
- document naming conventions for nodes, branches, checks, and routes

## Content Direction

Use a plausible undergraduate CS path:
- Programming Fundamentals
- Discrete Mathematics
- Data Structures
- Algorithms
- Computer Systems
- Databases
- Software Engineering
- Networks
- Operating Systems
- AI/ML Intro
- Security
- Capstone

First playable slice should focus on:
- Programming Fundamentals
- Discrete Math
- Data Structures
- Algorithms

## Done When

- a curriculum outline exists in this epic
- first playable slice is clearly bounded
- dependencies and branch structure are specified
- target node count for the first slice is set
- assessment type coverage plan exists
- content is ready for a seed/template implementation agent
