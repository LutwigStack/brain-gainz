# 02 Full Pixel Today

## Status

`parked`

## Goal

Переделать экран `Today` в полноценный `pixel-RPG command screen`, близкий по духу к выбранным референсам.

## Why

Сейчас `Today` решает продуктовую задачу, но визуально ощущается как обычный продуктовый экран. Для BrainGainz этого недостаточно: главный daily loop должен восприниматься как ритуал и игровой command center.

## Scope

- новый shell layout для `Today`;
- dominant focus block;
- quest/action rows;
- streak, skills to review, weakening branches;
- domain progress, near completion, recent unlocks;
- daily rituals и reflection widgets;
- avatar / apprentice panel;
- pixel framing, pixel typography и unified progress language.

## Done When

- `Today` больше не выглядит как generic dashboard;
- главный фокус, secondary actions и progression widgets собраны в единый command-screen layout;
- визуальный язык совпадает с зафиксированным `pixel design system`;
- screen дает ощущение ежедневного игрового ритуала, а не просто списка задач.
