# 02 Hide Author Mode From Learner Path

## Status

`planned`

## Goal

Keep author mode available but remove it from the learner's first path.

## Scope

- move `Настраиваю` out of the primary top shell in learner mode
- provide a secondary entry such as campaign settings, overflow menu, or explicit edit action
- make the entry clear enough for authors but quiet for learners
- ensure learner mode still hides create/connect/delete/archive controls
- ensure author mode remains fully reachable

## Done When

- new learner does not see author mode as a main choice
- author can still enter editing intentionally
- destructive controls remain protected
