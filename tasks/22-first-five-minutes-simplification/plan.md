# First Five Minutes Simplification Plan

## Order

1. Make Today start the focused lesson/check directly.
2. Move author mode out of the learner's first path.
3. Replace internal terms in learner-facing copy.
4. Simplify the top shell and side navigation for learner mode.
5. Tighten mobile first screen.
6. Run first five minutes browser QA.

## Main Risk

Do not remove author tools. Move them out of the learner's way.

## UX Checks

For the first five minutes, each screen should answer:

- what am I doing now?
- what is the one main button?
- what happens next?
- what can I ignore?

If a user must read several panels to choose the next action, the screen is still too complex.
