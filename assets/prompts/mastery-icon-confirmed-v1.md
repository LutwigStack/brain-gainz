# Mastery Icon: Confirmed V1

## UI Slot

- Surface: node card, route progress, Today reward marker.
- Slot: small mastery badge.
- Target size: 64x64, must remain readable at 20-24px.
- Format: PNG or SVG, transparent background.

## Prompt

Create a transparent-background compact game icon for confirmed mastery in a dark strategy learning interface. Symbol should feel like secured territory or verified knowledge: a small luminous seal, crystalline check mark, or fortified node marker. Pixel-art influenced, clean silhouette, high contrast, cyan and gold accent, no readable text, no numbers, no UI frame, no watermark.

## Negative Constraints

- No letters or words.
- No large decorative background.
- No complex multi-object scene.
- No generic app badge with rounded rectangle.

## Review Notes

The selected variant must be checked at 20px, 24px, and 64px. If it is not readable at 20px, reject it.
