# Opponent Portrait: Rival Architect V1

## UI Slot

- Surface: Today pressure panel.
- Slot: compact opponent portrait beside pressure meter.
- Target size: 192x192.
- Format: PNG, transparent background.

## Prompt

Create a transparent-background game portrait of a rival learning strategist called the Rival Architect. Dark retro-futuristic strategy-game style, pixel-art influenced but polished, strong silhouette, angular cloak or armor, subtle glowing planning instruments, analytical expression, science-fantasy mood, readable at small UI size, high contrast for dark interface panels. No readable text, no UI frame, no watermark, no logos.

## Negative Constraints

- No text or numbers.
- No gore.
- No photorealistic stock look.
- No full scene background.
- No decorative bokeh or gradient-orb background.

## Review Notes

Use this only after the Today pressure slot is ready for image rendering. Generate 2-4 variants, select one, then verify at 48px, 96px, and 192px.
